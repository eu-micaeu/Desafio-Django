# DESAFIO DJANGO

## O que precisa para rodar?

### Python 3.12.7

## Como rodar?

Na pasta raiz, execute os seguintes comandos ( Nessa ordem ) :

```
cd desafioDevDjqango
```

```
python manage.py runserver
```

## Para realizar os testes:

Na pasta raiz, execute os seguintes comandos ( Nessa ordem ) :

```
cd desafioDevDjqango
```

```
python manage.py test
```
